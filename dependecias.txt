Se usa python 3.8.10 - tutorial (https://www.youtube.com/watch?v=cfgdhakkfGY)
Para ejecutar el proyecto se usa el comando:
En Windows puede que sea un poco diferente la instalacion
-> python3 main.py

Este proyecto ocupa las librerias:
Analisis de textos
-> pip3 install --user -U nltk
-> pip3 install -U textblob

Matematicas y estadisticas
-> pip3 install numpy
-> pip3 install matplotlib

Extraccion de texto de pdf
-> pip3 install PyPDF2

Generacion de graficas
-> pip3 install wordcloud  (de dar problemas instalar pip install pillow==9.5-0)

Interfaces
-> pip3 install dearpygui

Guardado y restauracion de datos 
-> pickle (ya viene con python)

Notas: 
----La lectura de archivos con la funcion open en windows puede ser diferente a linux
